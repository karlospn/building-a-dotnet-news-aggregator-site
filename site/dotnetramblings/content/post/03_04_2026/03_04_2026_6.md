React Native Windows v0.82 is here, marking a major milestone: the legacy Paper architecture has been fully removed. All applications now run exclusively on the New Architecture (Fabric), and this release also unlocks XAML controls for community modules — so Windows apps can seamlessly mix native XAML controls with React components.
